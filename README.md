# 水戸地図

水戸地図はD3.jsやOpenLayersなどJavaScriptのライブラリを使ってデータビジュアライゼーションを試みる実験室です

### 使用する言語

- JavaScript
- Python

### 使用するライブラリ

- [D3.js]
- [OpenLayers]
- [Leaflet.js]
- [jQuery]
- [Bootstrap]
- [hugo]
- [Node.js]
- [GDAL/OGR][GDAL]
