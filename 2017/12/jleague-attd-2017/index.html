J.League 2017 All Games Attendance Visualization
2017年の明治安田生命Jリーグ全試合の観客数を可視化したチャートです。

上の画像は静止画ですがインタラクティブ版(PC推奨)もあります。

  • 縦軸: クラブ名（順位順）
  • 横軸: ホームゲーム観客数
  • 円の色
    • 緑: 勝ち
    • グレー: 引き分け
    • 赤: 負け
  • 点線: クラブ毎の平均観客数を繋いだもの
